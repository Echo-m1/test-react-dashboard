# Дашборд «Кадры / Техподдержка»

SPA на React для отдела кадров: аналитика на дашборде, справочник людей с фильтрами, детальная карточка человека с разделами и связями (семья, образование, паспорт, адреса).

## Стек

- **React 19** (JavaScript)
- **Vite** — сборка
- **Material UI (MUI)** — интерфейс, тема
- **Redux (RTK)** — состояние: список людей, фильтры, выбранный человек, агрегаты дашборда
- **React Router** — маршрутизация
- **Recharts** — графики на дашборде
- **Zod** — валидация форм

## Структура проекта

```
src/
  main.jsx           — точка входа
  App.jsx            — роутинг, инициализация данных
  components/        — общие компоненты (Layout, Error Boundary)
  pages/
    Dashboard/       — главная: метрики, графики (рождения, пол, заявки)
    PeopleList/      — картотека: таблица, фильтры, счётчик, пагинация
    PersonCard/      — карточка человека: вкладки, поля ввода, связанные таблицы (семья, образование, адреса)
  store/             — Redux: слайсы (people, filters), селекторы
  data/              — генератор тестовых данных (generatePeople)
  theme/             — тема MUI
  utils/             — утилиты, константы, маршруты (routes.js), валидация (personSchema)
```

## Запуск и сборка

1. Установка зависимостей: `npm install`
2. Запуск: `npm run dev` — приложение откроется по адресу из вывода Vite (обычно http://localhost:5173)
3. Сборка: `npm run build` — результат в папке `dist/`

Данные генерируются при первом открытии приложения (20–30 тыс. записей для теста).
